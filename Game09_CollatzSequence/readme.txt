Lothar Collatz tarafından 1932'de
keşfedilmiştir. Teorisi basittir ama
henüz matematiksel ispatı yoktur.

    n>0 iken

    n tek sayı ise n*3+1,
    n çift sayı ise n/2

şeklinde sonraki sayı bulunarak
ilerlendiğinde mutlak bir noktada
4,2,1,4 döngüsüne gelinir. Buna 
göre sıralama mutlaka 1 sayısına
ulaşır.

Örneğin n = 5 için.

5, 16, 8, 4, 2, 1, 4, 2, 1, 4

ya da

n = 12 için

6, 3 , 10, 5, 16, 8, 4, 2, 1, 4, 2, 1,
